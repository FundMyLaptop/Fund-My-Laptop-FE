3.1 Primary Button

Primary Button according to design style guide

Use the btn-fml-[primary|secondary|accent] class to apply any of the project colors colors to a button

Examples

Default styling

.btn-fml-primary

primary button with primary color

.btn-fml-secondary

primary button with secondary color

<button class="btn [modifier class]">A button</button>

3.2 Outlined Buttons

You can use the bootstrap utility classes(btn-outline-[color]) for outlined buttons The border-width property on the outlined buttons that use FML's colors have been set to 1.5px to match style guide

Examples

Default styling

.btn-outline-fml-accent

Outlined button with fml-accent color

.btn-outline-fml-secondary

Outlined button with fml-secondary color

<button class="btn [modifier class]">Outlined Button </button>
