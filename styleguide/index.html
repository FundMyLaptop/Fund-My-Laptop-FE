FundMyLaptop Mini-styleguide

Whaaaat, styleguides?

Styleguides can be seen as a roadmap for developers and designers to maintain consistency on their code and UI. Some styleguides are built in static HTML, but this is difficult to maintain. A better solution would be a living styleguide, which is auto-generated from the comments in your codebase.

What is contained here?

This is a mini documentation for some of the bootstrap customizations made to the frontend project files for the FundMyLaptop project.

The customizations here would enable frontend developers on this project to avoid repeating some basic styling that is consistent across the project(font-family, buttons colors and style, e.t.c).

It would also help to enforce consistency on our most used UI elements and components and help the frontend team adhere to the styleguides provided by designers.

Bootstrap's customized defaults

Below are styles you don't have to repeat in your style rules as you contribute to this project.

  • font-family - the body default font-family has been set to the project's main font - DM Sans
  • h1, h2, h3 - font sizes for these heading have been set to the values provided on the design styleguide

*All other customizations are contained in their respective sections in the documentation menu
